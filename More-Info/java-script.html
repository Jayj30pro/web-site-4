JavaScript is often confused with another programming language called Java, but they are different languages. Brendan Eich created JavaScript in 1995. Eich was a programmer at Netscape Communications Corporation when he created a scripting language called Mocha. JavaScript was created for the Netscape navigator and became known as LiveScript. The name changed to JavaScript to show that Netscape's browser supported Java. In 1996, the ECMA (European Computer Manufacturers Association) adopted JavaScript. The ECMA released version 2 in 1998 and version 3 in 1999, which became the standard.
