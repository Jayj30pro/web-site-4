Java was created at Sun Microsystems by a team led by James Gosling. The goal was to create a language that would allow electronic devices to communicate with one another. The research started in 1991, and along the way, the focus changed to the upcoming world wide web. Java was released in 1995 and revolutionized the industry. Because Java compiles into bytecode and then interprets, one code can run on several machines without modifications to the code.
