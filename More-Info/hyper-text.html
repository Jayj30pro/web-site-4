The first version of HTML was released in 1993 by Tim Berners-Lee. There have been many versions of HTML, but HTML 4.01 became the standard in December 1999. In 2000 XHTML became the official standard with some updates along the way. In 2011 drafts began for HTML5. The new version would add audio, video, and canvas tag, allowing for exciting possibilities. HTML 5 became the official standard in 2014.
